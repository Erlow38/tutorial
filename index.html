Tutorial: Create a React project with a form component

Prerequisites

~30/45 minutes.

Medium difficulty.

If you want to go faster by ignoring non-essential information, only follow the pink color instructions.

Before starting this tutorial, a basic knowledge of HTML, CSS, and JavaScript is a plus.

You must have an IDE, for example Visual Studio Code.

Step 0: What is React

React, also known as React.js or ReactJS, is an open-source JavaScript library for building user interfaces. It was developed by Facebook and is commonly used for creating dynamic and interactive web applications. React is particularly well-suited for building single-page applications and is often used in conjunction with other libraries and frameworks, such as Redux for state management.

Step 1: Install Node.js and npm

Before creating a React project, make sure you have Node.js and npm installed on your system. You can download them from the official Node.js website.

Step 2: Install Create React App

To simplify the process of creating a React project, you can use Create React App, a tool that automatically sets up a React development environment.

To install it, open your terminal and run the following command:

npx create-react-app my-react-project

Make sure to replace "my-react-project" with the name you want for your project.

Step 3: Start the development server

Once the installation is complete, navigate to the project folder and run the following command:

cd my-react-project
npm start

This will start the development server and open your project in your default browser.

Step 4: Basic File Structure

Let's take a look at the basic file structure of a React project created with Create React App.

First, you have the node_modules folder, which contains all the dependencies of your project.

Then, you have the public folder, which contains the index.html file, the favicon.ico file, and the manifest.json file.

Finally, you have the src folder, which contains all the source code of your project.

For now, we will only focus on the src folder.

Step 5: The index.js file

Let's start by taking a look at the index.js file.

This file is the entry point of your React application. It is responsible for rendering the App component in the root element of the index.html file.

By default, the App component is a simple component that displays the React logo and a welcome message.

Let's change the welcome message to "Hello World!"

import React from 'react'; function App() { return ( <div className="App"> <header className="App-header"> <img src="logo.svg" className="App-logo" alt="logo" /> <p> Welcome to React </p> <a className="App-link" href="https://reactjs.org" target="_blank" rel="noopener noreferrer" > Learn React </a> </header> </div> ); } export default App;

Step 6: Understanding React Components

React is a component-based library. This means that you can create components and reuse them throughout your application.

There are two types of components in React: functional components and class components.

Functional components are simple functions that take props as arguments and return a React element.

Class components are more complex components that can have state and lifecycle methods.

For now, we will only focus on functional components.

Let's create a new component called Form and display it in the App component.

Step 7: The Form component

Let's create the Form.jsx file in the src folder.

Then, in the Form.jsx files, create a functional component called Form and export it.

 import React from 'react'; function Form() { return ( <div> <h1>Form</h1> </div> ); } export default Form;

Step 8: Complete the Form file

Here's a simple example of a form component with two inputs and a submit button. You can replace the return statement in Form.jsx files with this:

<form id='form' onSubmit={handleFormSubmit}> <div> <label>Name:</label> <input type="text" name="name" value={formData.name} onChange={handleInputChange} /> </div> <div> <label>Email:</label> <input type="email" name="email" value={formData.email} onChange={handleInputChange} /> </div> <button type="submit">Submit</button> </form>

Step 9: Add a callback function

Still in the Form.jsx file, add this before the return. This will allow you to call a callback function when you send the form.

import React, { useState } from 'react'; function Form({ onSubmit }) { const [formData, setFormData] = useState({ // Initialize your form fields here name: " ", email: " ", }); const handleFormSubmit = (e) => { e.preventDefault(); // Perform any form data validation here onSubmit(formData); // Call the callback function with form data document.getElementById("form").style.display = "none"; // Hide the form when submit }; const handleInputChange = (e) => { const { name, value } = e.target; setFormData({ ...formData, [name]: value }); }; }

Step 10 : Adding style

To make our form hidden and look a little prettier, in the App.css file, add the following style:

form {   max-width: 400px;   margin: 0 auto; } form div {   margin-bottom: 15px; } form label {   display: block;   font-weight: bold;   margin-bottom: 5px; } form input {   width: 100%;   padding: 8px;   box-sizing: border-box;   border: 1px solid #ccc;   border-radius: 4px; } form button {   background-color: #4caf50;   color: white;   padding: 10px 15px;   border: none;   border-radius: 4px;   cursor: pointer; } form button:hover {   background-color: #45a049; }

Step 11: Add the Form component in the App component

Now that we have created the Form component, let's add it in the App component.

To do this, we need to import the Form component in the App component and add it in the return statement like this.

import React from 'react'; import Form from './Form'; function App() { // Define a callback function to handle form submission const handleFormSubmit = (formData) => { // Do something with the form data, e.g., send it to a server console.log('Form data submitted:', formData); }; return ( <div className="App"> <header className="App-header"> <img src="logo.svg" className="App-logo" alt="logo" /> <p>Welcome to React</p> <a className="App-link" href="https://reactjs.org" target="_blank" rel="noopener noreferrer" > Learn React </a> </header> <Form onSubmit={handleFormSubmit} /> {/* Pass the callback function to the Form component */} </div> ); export default App;

Step 12: Add a button for display the form

Now that we have added the Form component in the App component, let's add a button to display the form.

To do this, in the App.jsx file, add the following code before the return statement.

const handleShowForm = () => { document.getElementById("form").style.display = "block"; };

In the return statement, add the following code to display the form when the button is clicked.

<button onClick={handleShowForm}>Show Form</button>

Step 13: Congratulation!

Congratulations! You have successfully created your first React application whith a beautiful form.
